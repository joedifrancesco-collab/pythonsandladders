# pythonsandladders
Pythons & Ladders is a text-based Snakes & Ladders game written in Python.

## Features
- 1 to 3 players
- Name prompts for each player — press Enter to use a default name (Player 1, 2, 3)
- All players finish; final standings reflect the order each player reached square 100
- Single space bar keypress to roll (no Enter required)
- Autoplay mode — press **A** at any prompt to let the computer finish the game automatically
- Die rolls, movement, ladders, and pythons
- Exact-roll rule to win on square 100
- End-of-game standings with winner, placements, and roll counts

## Run
From the repository folder:

```powershell
python game.py
```

## Controls
| Key | Action |
|-----|--------|
| `Space` | Roll the die / advance to next turn |
| `A` | Enable autoplay — computer completes the game automatically |
| `Q` | Quit at any prompt |

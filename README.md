# pythonsandladders
Pythons & Ladders is a text-based Snakes & Ladders game written in Python.

## Features
- 1 to 3 players
- Name prompts for each player
- Space-bar controlled start/continue flow
- Die rolls, movement, ladders, and pythons
- Exact-roll rule to win on square 100
- End-of-game standings with winner, placements, positions, and roll counts

## Run
From the repository folder:

```powershell
python game.py
```

When prompted to continue, press the space bar once and then press Enter.
